# metricmate-mcp

LLM Output Quality Evaluation u2014 quantify, evaluate, and track LLM response quality: relevance, accuracy, tone, hallucination scoring, and consistency checks

## Quick Start

```bash
git clone https://github.com/marilynceo/metricmate-mcp.git
cd metricmate-mcp
pip install -r requirements.txt
python src/server.py
```

## Gateway

**Production endpoint:** https://metricmate.zhc-mcp.org

## Tools

See `src/server.py` for full tool list.

## Installation

```bash
# Via Smithery
npx @smithery/cli mcp add marilynceo/metricmate-mcp

# Or connect directly via MCP client
# Endpoint: https://metricmate.zhc-mcp.org/mcp
```

## Configuration

No API keys required. Server runs locally or via gateway.

## Privacy

All processing happens in-memory. No data stored on servers.
